# 🗳️ Democracy Guide | Your Election Assistant

Democracy Guide is a modern, interactive web application designed to simplify the voting process. It provides a step-by-step journey through voter registration, candidate research, early voting, and more, featuring an AI-powered assistant to answer your questions in real-time.

![Preview of Democracy Guide](https://img.shields.io/badge/Status-Active-brightgreen)
![License](https://img.shields.io/badge/License-MIT-blue)

## ✨ Features

- **Interactive Election Journey**: A vertical timeline guiding users from registration to election results.
- **Smart Voter Assistant**: An integrated chat interface that provides instant answers about registration, voting methods, and key dates.
- **Dynamic Content Cards**: Detailed information and action checklists for each step of the election process.
- **Responsive Design**: A sleek, premium UI built for both desktop and mobile users.
- **Rich Aesthetics**: Dark mode interface with glassmorphism effects and smooth animations.

## 🛠️ Technologies Used

- **Frontend**: HTML5, CSS3 (Vanilla CSS)
- **Logic**: JavaScript (ES6+)
- **Typography**: [Inter](https://fonts.google.com/specimen/Inter) via Google Fonts
- **Icons**: [Font Awesome 6](https://fontawesome.com/)
- **Animations**: CSS Keyframes

## 🚀 Getting Started

To run this project locally, follow these simple steps:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/motalib-code/swasti-prompt-war.git
   ```

2. **Open the project**:
   Simply open the `index.html` file in any modern web browser.

3. **Explore**:
   - Click on the steps in the **Election Journey** sidebar to see detailed guides.
   - Use the **Voter Assistant** chat to ask questions like *"How do I register?"* or *"When is Election Day?"*.
